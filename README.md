# 📊 Marketing Campaign Analysis & ROI Dashboard – Power BI

## ✨ About This Project

This Power BI dashboard was developed to analyze marketing campaign performance and measure Return on Investment (ROI) across regions, customer segments, and demographics.

The goal was to transform raw marketing data into actionable insights that support data-driven decision-making.

This dashboard helps answer:

- Which campaigns deliver the highest ROI?
- How do impressions convert into clicks and conversions?
- Which customer segments drive the most revenue?
- How does performance vary by gender and age group?
- 
## 📈 Dashboard Highlights

- KPI Cards (Impressions, Clicks, Conversions, Sales, ROI)
- ROI Analysis for Top Performing Campaigns
- Sales by Gender (Donut Chart)
- Customer Segment Analysis
- Sales by Age Range
- Location-wise Drilldown Table
- Campaign Funnel Performance Insights

## 📊 Key Metrics Tracked

- Impressions
- Clicks
- Conversions
- Ad Spend
- Revenue
- ROI (Return on Investment)
- Customer Demographics
